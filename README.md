# Task Manager Application

This is a simple task manager project built using Laravel and React.

It allows users to register, login and manage their tasks with features like create, update, delete and filtering.

## Features

* User authentication (Login / Register)
* Create, update and delete tasks
* Each user sees only their own tasks
* Task status (pending / completed)
* Pagination and filtering
* Clean UI

## Tech Stack

Frontend: React (Vite), Axios
Backend: Laravel 12, MySQL

---

## Setup

### Backend

cd backend
composer install
php artisan migrate
php artisan serve

### Frontend

cd frontend
npm install
npm run dev
